
# Hardware Benchmarking

This branch contains the scripts and data for the hardware benchmarking.

### /benchmark
Contains developer scripts for testing the Braket API.

### /benchmark2
Contains the actual benchmark scripts for the quantum and classical device benchmarks.

### /process_csv
Contains the script for normalizing the raw throughputs, and the normalized throughput results.
All the results are stored in csvs in `/process_csv`.
